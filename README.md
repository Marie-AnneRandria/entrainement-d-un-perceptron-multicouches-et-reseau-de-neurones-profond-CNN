# Entraînement d'un perceptron multi-couches et réseau de neurones profond CNN

## Objectif principal
Ce projet vise à étudier un réseau de neurones de type Perceptron MultiCouches sur le cas de vente en ligne. Il vise également à étudier un réseau de neurones profond de type convolution (CNN) pour classer les images. 

## Outils utilisés

* Scikit-Learn
* Keras
* TensorFlow
